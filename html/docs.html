Installation:
  • Download the Printer's binary zip file
  • Unzip Printers to C:\Printers or some other suitable directory (e.g. c:\windows or c:\winnt)
  • Right click on the Start Button and select Explore
  • In the right hand pane,
    • double click on programs
    • double click on StartUp
    • Right click in the white area
    • Select New and then Shortcut
    • Enter the location of the Printers.exe file
    • Enter a Name for the Shortcut
  • Close Explorer
