About Printers

Printers is an application that was written to make it easy to change the default printer in Windows 95.  Until it was written, there were no easy methods to quickly view or change the default windows printer.

Printers was originally written around 1998 and has been in constant use in a corporate environment ever since.  

Printers is licenced under the GNU General Public Licence (GPL) and the Download link to the left provides access to both the application and the source code.
